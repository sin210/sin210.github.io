| 层级 | 技术成果 | 自主可控说明 |

| **1. 芯片层** | **Ascend 910B**（昇腾910B） | 采用7nm工艺，支持FP16/FP32/BF16等多精度计算，算力达256 TFLOPS（FP16），**完全由华为自研**，实现“中国芯”算力突破。 |

| **3. 操作系统层** | **华为云 EulerOS**（欧拉操作系统） | 基于Linux内核，专为服务器、云、边缘场景优化，支持容器化、微服务架构，**已通过国家信息安全认证**，实现系统级自主可控。 |
| **4. 框架层** | **MindSpore**（昇思） | 华为自研的AI框架，支持端-边-云协同推理与训练，具备自动微分、图优化、分布式训练能力，**代码完全开源，已纳入Apache 2.0协议**。 |
| **5. 应用层** | **昇腾AI云服务（Ascend Cloud Service）** | 提供AI训练、推理、模型托管、自动调参等全链路服务，支持主流大模型（如盘古、华为云百模）快速部署，**所有服务均运行在国产化基础设施之上**。 |
